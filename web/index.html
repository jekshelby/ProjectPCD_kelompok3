I Image

Fitur Yang Dapat Dipakai

Compress Image

Compress image bertujuan untuk memperkecil ukuran file gambar tanpa mengurangi kualitasnya secara signifikan.

Greyscale

Greyscale image bertujuan untuk mengubah warna gambar menjadi hitam-putih (Abu-abu).

Remove Background

Remove background image bertujuan untuk menghapus background dari image.

Super Resolution

Super Resolution image bertujuan untuk meningkatkan resolusi gambar, menghasilkan gambar yang lebih tajam dan lebih jelas.

Crop Image

Crop image bertujuan untuk memangkas bagian gambar yang diinginkan.
